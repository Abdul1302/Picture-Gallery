# Cloudinary Image Gallery

## Overview
This project is a picture gallery implemented using Cloudinary to manage and display images. It includes several key features such as a gallery showcasing images from Unsplash, a picture album for organizing images into folders, and a favorites page for saving preferred pictures. Additionally, there are interactive icons for editing, favoriting, and organizing images into folders.

## Features

### 1. Gallery
The gallery page displays images sourced from Unsplash. Users can browse through a collection of diverse images.

### 2. Picture Album
In the picture album, users can organize their images into folders. Clicking on the folder icon for a picture opens a small form where users can assign the picture to a specific folder.

<img width="960" alt="7" src="https://github.com/Abdul1302/Project02-Cloudinary-Image-Gallery/assets/106179686/cf6bab83-0530-4243-b170-eea90b09194e">

### 3. Favorites
The favorites page allows users to save their preferred pictures. Clicking on the favorite icon for a picture adds it to the favorites collection.

<img width="960" alt="2" src="https://github.com/Abdul1302/Project02-Cloudinary-Image-Gallery/assets/106179686/f7d0219c-29c2-4f9f-a9b0-9b1386f9f4f4">


## Interactive Pages
Each page in the gallery have their own function:

### Edit Page: 
Clicking on the edit icon redirects users to a new page where they can edit the selected picture.

<img width="960" alt="6" src="https://github.com/Abdul1302/Project02-Cloudinary-Image-Gallery/assets/106179686/8fe4001f-9a37-4b70-a435-db960b35d5b6">

### Favorite Page:
Clicking on the favorite icon adds the selected picture to the favorites page.

<img width="960" alt="4" src="https://github.com/Abdul1302/Project02-Cloudinary-Image-Gallery/assets/106179686/ceabc851-7c78-4782-ab1f-54bb7b8015b7">

### Folder Page:
Clicking on the folder icon opens a form allowing users to assign the picture to a specific folder in the picture album.

<img width="960" alt="3" src="https://github.com/Abdul1302/Project02-Cloudinary-Image-Gallery/assets/106179686/f865f7bf-ee90-45f2-97f4-69de2665acbc">

### Upload:
The main picture gallery page features an "Upload" button, allowing users to add custom images to the cloud and include them in the gallery. Upon clicking the button, users can select an image from their local machine or from given cloudinary function of uploading from unsplash images and upload it to Cloudinary.
